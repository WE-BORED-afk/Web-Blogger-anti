# Design System: Web Blogger - บล็อกส่วนตัวของฉัน

## 1. Overview

**Creative North Star: "The Minimalist Journal" (บันทึกส่วนตัวมินิมอล)**

ระบบการออกแบบนี้เน้นความเป็นระเบียบ ความสะอาดตา และความประณีตของการจัดวางตัวพิมพ์และการเว้นระยะห่าง เพื่อให้เว็บบล็อกส่วนตัวเป็นเหมือนสมุดบันทึกที่สวยงาม อ่านสบายตา และกลมกลืนกับธรรมชาติ โดยใช้สีโทนเขียวเสจผ่อนคลายตัดกับเนื้อครีมทรายอุ่น

การออกแบบนี้ยึดหลักเลย์เอาต์หน้าเดียวที่ผู้ใช้อ่านรายละเอียดบทความในหน้านั้นๆ ได้ทันทีโดยไม่ต้องโหลดเว็บใหม่ และไม่มีระบบคอมมูนิตี้มารบกวนความเป็นส่วนตัว

**ลักษณะการออกแบบที่สำคัญ:**
- การใช้สีพาสเทลเอิร์ธโทนเพื่อให้ความรู้สึกผ่อนคลายและเข้ากับธรรมชาติ
- การเล่นระดับมิติด้วยเอฟเฟกต์กระจกฝ้า (Glassmorphism) บนการ์ดเนื้อหา
- การจัดลำดับความหนาบางของตัวอักษรไทย-อังกฤษอย่างประณีต
- การซ่อนเครื่องมือควบคุมผู้เขียน (เช่น กล่องใส่รูปปก/ลิงก์รูปภาพเพิ่มเติม) ออกจากการมองเห็นของบุคคลทั่วไปโดยเด็ดขาด

## 2. Colors

ชุดจานสีอ้างอิงแนวคิด "Sage & Sand" ที่นำพาความรู้สึกเงียบสงบในป่าเสจผสมผสานความอุ่นของชายหาดสีเม็ดทราย

### Primary
- **Sage Green** (#9AB17A): สีเขียวเสจหลัก ใช้เป็นสีแบรนด์, ลิงก์ที่ไฮไลท์, และปุ่มเปิดโหมดแก้ไขหลัก

### Secondary
- **Light Sage** (#C3CC9B): สีเขียวเสจอ่อน ใช้สำหรับการตอบสนองของเมาส์ชี้ (Hover) และเส้นแบ่งการแก้ไข

### Neutral
- **Cream Beige** (#E4DFB5): สีครีมเบจธรรมชาติ ใช้เป็นสีพื้นหลังของเว็บ
- **Warm Sand** (#FBE8CE): สีทรายอุ่นประกายประกบ
- **Translucent Sand** (#FBE8CE73 หรือ `rgba(251, 232, 206, 0.45)`): สีทรายโปร่งแสงผสมกระจกฝ้า ใช้เป็นพื้นหลังของการ์ดและส่วนหัว
- **Dark Olive** (#3A4730): สีเขียวมะกอกเข้มมาก ใช้เป็นสีตัวอักษรหัวข้อและลิงก์หลักเพื่อให้คอนทราสต์ชัดเจน
- **Earthy Charcoal** (#2D3528): สีเทาถ่านปนดิน ใช้เป็นสีเนื้อหาหลัก (Body text)
- **Muted Green-Gray** (#5E6955): สีเทาเขียวหม่น สำหรับตัวหนังสือย่อยและข้อมูลระบุวันที่/เวลาอ่าน

**กฎการจำกัดสี (The Restrained Palette Rule).**
ห้ามใช้สีสันภายนอกอื่นๆ นอกเหนือจากชุดสีที่กำหนดไว้นี้เด็ดขาด เพื่อคงความสงบนุ่มนวลของแบรนด์ส่วนบุคคลของบล็อกนี้

## 3. Typography

**Display Font:** Outfit (ใช้สำหรับภาษาอังกฤษและตัวเลข)
**Body Font:** Prompt (ใช้สำหรับภาษาไทยและเนื้อหาหลัก)

### Hierarchy
- **Display (H1)** (Bold, 2.2rem, 1.4): ใช้สำหรับหัวข้อหลักของโพสต์ในหน้าอ่านรายละเอียด
- **Headline (H2)** (Bold, 1.8rem, 1.4): ใช้สำหรับหัวข้อโปรไฟล์และหัวข้อในการ์ดโพสต์บนฟีดหลัก
- **Body** (Regular, 1.05rem, 1.7): ใช้สำหรับเนื้อหาการอ่านบล็อกและความยาวบรรทัดสูงสุดไม่เกิน 75ch เพื่อความง่ายต่อการกวาดสายตา
- **Label** (Medium, 0.85rem, 1.1): ใช้สำหรับข้อมูลย่อย เช่น วันที่ เวลาอ่าน และแท็กต่างๆ

**กฎการปรับสัดส่วนข้อความยาว (The Text-Wrap Rule).**
ต้องใช้ `text-wrap: balance` เสมอในส่วนหัวข้อโพสต์หลักเพื่อกระจายน้ำหนักคำภาษาไทยไม่ให้ตกหล่นทีละพยางค์ และใช้ `text-wrap: pretty` ในพารากราฟเนื้อหา

## 4. Elevation

การสร้างมิติมุ่งเน้นการใช้ความแบนราบ (Flat) ประสานกับวัสดุประเภทกระจกโปร่งแสงฝ้า (Glassmorphism) และระดับแสงเงาที่บางเบาที่สุดเพื่อหลีกเลี่ยงดีไซน์แบบเว็บซอฟต์แวร์บริการ (SaaS) ทั่วไป

### Shadow Vocabulary
- **Ambient Glow** (`box-shadow: 0 8px 32px 0 rgba(58, 71, 48, 0.08)`): ใช้สร้างมิติแบบแผ่ขยายใต้อุปกรณ์บล็อกและการ์ดเนื้อหา
- **Interactive Rise** (`box-shadow: 0 16px 45px 0 rgba(58, 71, 48, 0.12)`): เงาที่ทอดตัวลึกขึ้นเล็กน้อยเมื่อผู้ใช้งานเอาเมาส์ไปชี้ที่การ์ด

**กฎความแบนราบโต้ตอบ (The Flat-by-Default Rule).**
องค์ประกอบหลักของเว็บบล็อกต้องเรียบราบขอบบางเมื่อจอดนิ่ง และจะนูนลอยเล็กน้อยพร้อมแอนิเมชันก็ต่อเมื่อมีปฏิสัมพันธ์จากผู้ใช้เท่านั้น

## 5. Components

### Buttons
- **Shape:** มนโค้งแคปซูล (`border-radius: 50px`)
- **Primary Button:** ใช้พื้นหลังสีเขียวเสจ ตัวหนังสือสีขาว และ padding `0.6rem 1.2rem`
- **Secondary / Cancel Button:** ใช้พื้นหลังสีโปร่งแสงอ่อนๆ พร้อมการชี้เมาส์เปลี่ยนโทนสีเป็นครีม

### Cards / Containers
- **Corner Style:** มุมโค้งกว้างนุ่มนวล (`border-radius: 16px`)
- **Background:** พื้นหลังกระจกฝ้าทรายโปร่งแสง (#FBE8CE73) ผสานฟิลเตอร์เบลอฉากหลัง (`backdrop-filter: blur(12px)`)
- **Border:** ขอบแก้วสว่างบางเฉียบ (`1px solid rgba(255, 255, 255, 0.4)`) เพื่อขับเน้นกรอบให้คมชัดบนพื้นหลังสีเบจ

### Inputs / Fields
- **Style:** โหมดแก้ไขใช้เส้นประสีเขียวเสจเป็นกรอบแสดงพื้นที่ที่พิมพ์แก้ไขได้
- **Edit Panel:** พาเนลจัดการใส่รูปปกหรือรูปเพิ่มเติมจะเป็นสีขาวทึบขอบประ และจะถูกตั้งค่าเป็น `display: none` ในโหมดการอ่านปกติ

## 6. Do's and Don'ts

### Do:
- **Do** ใช้รหัสสีคู่ตรงตามจานสี "Sage & Sand" และห้ามละเลยความคมชัดของสีตัวอักษรกับพื้นหลัง (ความคมชัดขั้นต่ำต้อง ≥ 4.5:1)
- **Do** ให้ความสำคัญกับการซ่อนพาเนลจัดการรูปภาพและการเพิ่มรูปภาพในหน้าแก้ไขไม่ให้ออกมาโผล่ที่หน้าแสดงผลปกติ (หน้า blog)
- **Do** ใช้การเฟดแอนิเมชันที่ราบรื่น (Ease-out-quart) ในการเปลี่ยนหน้าอ่านรายเรื่อง

### Don't:
- **Don't** ใช้เส้นขอบสีหนาๆ กั้นเฉพาะขอบด้านข้างการ์ด (Side-stripe borders)
- **Don't** ปล่อยให้ตัวหนังสือเป็นสีเทาจาง (Low contrast gray) ที่จมไปกับพื้นหลังเบจ
- **Don't** ทำระบบไลก์หรือช่องคอมเมนต์ที่เป็นการชักชวนสังคมออนไลน์ (No community features)
- **Don't** ใส่แอนิเมชันขยายขนาดภาพโปรไฟล์หรือภาพหน้าปกตอนเอาเมาส์ชี้โดยตรง (ห้ามขยายรูปภาพเด็ดขาด ให้ใช้วิธีขยายเงาหรือสีขอบของการ์ดแทน)
- **Don't** นำฟีเจอร์สลับโหมดมืด (Dark Mode Toggle) กลับเข้ามาใช้งาน เพื่อควบคุมแบรนด์ให้เน้นโทนอบอุ่นตามความประสงค์หลัก
